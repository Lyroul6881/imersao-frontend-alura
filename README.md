# 🎬 Netflix Interface - Imersão Front-End Alura

Este projeto é uma reprodução da interface de seleção de perfis da Netflix, desenvolvido durante a **Imersão Front-End com IA** da [Alura](https://www.alura.com.br). O objetivo foi consolidar conhecimentos fundamentais de desenvolvimento web e explorar o potencial de ferramentas de IA no fluxo de trabalho de um desenvolvedor.

## 🔗 Demonstração
Você pode visualizar o projeto online aqui: 
👉 **[https://lyroul6881.github.io/imersao-frontend-alura/]**

---

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização, layout (Flexbox) e efeitos de hover.
* **JavaScript:** Lógica básica para interação de perfis.
* **Copilot/IA:** Auxílio na otimização do código e resolução de bugs.

## 🧠 O que eu aprendi
Durante as aulas ministradas por **Guilherme Lima**, **Thamiris Adriano** e **Fernanda Degolin**, foquei em:
1.  **Fidelidade Visual:** Como replicar um design de uma grande plataforma (Netflix).
2.  **Responsividade:** Garantir que a interface se comporte bem em diferentes telas.
3.  **Deploy:** O processo de tirar o código da máquina local e colocá-lo no ar via GitHub Pages.
